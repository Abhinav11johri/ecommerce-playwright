# Playwright Automation Framework - Ecommerce (Amazon)

This is an industry-level UI Automation framework built using **Playwright**, **TypeScript**, and the **Page Object Model (POM)** design pattern. It features custom fixtures, automatic failure screenshots, and data-driven testing.

## 🚀 Key Features
* **Page Object Model (POM):** Clean separation between page elements and test logic.
* **Custom Fixtures:** Simplifies test setup and provides automatic failure screenshots.
* **Data-Driven:** Uses JSON for managing test data (URLs, search terms, etc.).
* **Automatic Reporting:** Generates detailed HTML reports for every run.
* **CI/CD Ready:** Includes configuration for GitHub Actions.
